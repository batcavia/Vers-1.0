# Vers

Vers is een Expo React Native MVP voor progressieve Bijbeltekst-memorisatie.

## Scope

- Essentials-bundel met Statenvertaling als juridisch veilige demo-inhoud
- Structuur voorbereid op latere vertalingen via licentie of gebruikersimport
- Geen accounts
- Geen database
- Geen spraakherkenning
- Geen scores, coins, streaks of leaderboards
- Cursusmenu op de startpagina met thema's die de gebruiker kan kiezen en herhalen
- Cursussen met korte niveaus en een eenvoudige badge bij afronding
- Bijbeltekstcursussen rond rust, gebed, hoop, moed en woorden van Jezus
- Bijbelkennis-cursus om alle Bijbelboeken in volgorde te leren
- Korte interactieve lesflow met meerdere teksten tegelijk
- Missende woorden plek voor plek afvinken, zonder volledige tekst overtypen
- Eerste-letter en kettingvoorbereiding als latere stappen
- Warme, Duolingo-geinspireerde UI zonder externe assets
- Oude monnik in bruine habijt als rustige coach

## Starten

```bash
npm install
npm run start
```

De Expo SDK 57 projectconfig blijft leidend. De app gebruikt Expo Router via `app/src/app/index.tsx`.
